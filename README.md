# よかったカードおしゃべり機

発達障害のある子どもの視覚支援や、絵カードを使ったコミュニケーション支援を想定したWebアプリです。

## できること

- カテゴリー別の絵カード表示
- カードをトークバーに並べる
- トークバーの内容を日本語で読み上げる
- 自作カード作成
- 画像選択
- 文字の上・下配置
- 印刷用PNG作成
- スマートフォン・タブレットでの利用
- ホーム画面追加に対応

## GitHub Pagesで公開する方法

1. GitHubで新しいリポジトリを作ります。
2. このフォルダの中身をリポジトリにアップロードします。
3. GitHubのリポジトリ画面で `Settings` を開きます。
4. 左側の `Pages` を開きます。
5. `Build and deployment` の `Source` を `Deploy from a branch` にします。
6. `Branch` は `main`、フォルダは `/(root)` を選びます。
7. `Save` を押します。
8. 数分後、Pages画面に公開URLが表示されます。

公開URLはだいたい次の形になります。

```text
https://ユーザー名.github.io/リポジトリ名/
```

## 注意

自作カードの画像は、使っている端末のブラウザ内に保存されます。別のスマホやタブレットでは、同じ自作カードをもう一度作成する必要があります。
